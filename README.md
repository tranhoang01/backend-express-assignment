# Backend Framework Practice Assignment - Express

## 1. 프로젝트 소개

이 프로젝트는 백엔드 프레임워크 실습 과제를 위해 작성된 예제입니다.  
Express를 사용하여 Task(할 일) 관리용 RESTful API를 구현했습니다.

요구사항:
- POST / GET / PUT / DELETE 각각 최소 2개 API 구현 (총 8개 이상)
- Middleware 구현 (요청 로그, 404 처리, 에러 처리)
- 다양한 HTTP 응답 코드 사용 (2xx, 4xx, 5xx)
- 표준화된 응답 포맷 사용

---

## 2. 기술 스택

- Node.js
- Express
